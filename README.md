コンセプト

「8パズル」を自動で解くアルゴリズムを、視覚的にわかりやすく体験できるPythonアプリケーションです。

幅優先探索（BFS: Breadth-First Search）を用いて、初期配置からゴール状態までの最短手順を求め、その過程をアニメーションで表示します。

概要

3×3のスライドパズル（8 Puzzle）に対して、以下の機能を実装しています。

初期配置を設定
幅優先探索（BFS）による最短解探索
解の手順を自動再生
現在の手数を表示
完成時に「Done!」を表示

空白マス（0）を利用して数字タイルを移動させ、最終的に以下の状態を目指します。

幅優先探索（BFS）



幅優先探索を用いることで、必ず最短手数で解けるようにしました。

解の経路を保持

単なる「解けた / 解けない」だけでなく、各状態をリストとして保存し、アニメーション表示に利用しています。
（どのような手順でパズルを解いたのかを保存しています。）

視覚的なアニメーション

Pyxelを用いて10フレームごとに1手ずつ再生し、解法の流れを確認できます。

タイルごとの色分け

数字ごとに異なる色を設定し、視認性を向上させています。

手数の表示

現在の手数を画面下部に表示し、進行状況がわかるようにしました。
